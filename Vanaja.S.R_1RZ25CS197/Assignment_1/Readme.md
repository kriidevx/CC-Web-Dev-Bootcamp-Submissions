# What I built:
 I have built a simple, elegant Vanaja's To-Do list app where I can add my incompleted tasks into it. I have created it in such a way that it helps me keep track of my daily activities and make my day more produvtive.

# How to run it:
 Step:1 Open the dashboard
 Step:2 Type the task to be added in the empty box
 Step:3 Click on the add button
 Step:4 It appears under tasks to be completed column
 Step:5 If task is completed, click on the check box
 Step:6 Delete the task if not required

 I was very much interested in doing such things, I thoroughly enjoyed this process.
